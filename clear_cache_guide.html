🔄 Hướng dẫn Clear Cache Browser

⚠️ Tình huống:

Bạn vẫn thấy giao diện cũ tại https://aivannang.com

Nguyên nhân: Browser đang cache version cũ, chưa load version mới với UI cải tiến.

📱 Cách 1: Hard Refresh (Nhanh nhất)

Trên Windows:

  • Mở https://aivannang.com
  • Nhấn Ctrl + F5 hoặc Ctrl + Shift + R
  • Hoặc nhấn F12 → click chuột phải vào nút Refresh → chọn "Empty Cache and Hard Reload"

🧹 Cách 2: Clear Cache Thủ công

Chrome/Edge:

  1. Nhấn Ctrl + Shift + Delete
  2. Chọn "All time" hoặc "Last 24 hours"
  3. Tick "Cached images and files"
  4. Click "Clear data"
  5. Reload https://aivannang.com

Firefox:

  1. Nhấn Ctrl + Shift + Delete
  2. Chọn "Cache"
  3. Click "Clear Now"
  4. Reload https://aivannang.com

🔍 Cách 3: Kiểm tra trong Incognito

  • Mở Incognito/Private browsing (Ctrl + Shift + N)
  • Vào https://aivannang.com
  • Nếu thấy giao diện mới → Vấn đề là cache
  • Nếu vẫn cũ → Vấn đề deployment

✅ Sau khi clear cache, bạn sẽ thấy:

  • Admin Dashboard → Thời hạn Tab: UI mới với nút +/- cho số ngày
  • Quick buttons: +1d, +7d, 30d, ∞, Ban
  • Modal: Input đẹp với nút tròn cộng/trừ
  • Hiển thị: "X ngày = Y giờ"

🚨 Nếu vẫn không work:

Thử truy cập trực tiếp deployment URL:

https://aivannang-4q5hx326a-khanhs-projects-3f746af3.vercel.app

Nếu URL này work → Vấn đề domain. Nếu không → Vấn đề deployment.
